# basic-auth-api — Authn vs Authz Board
Simple api with basic authentication and api key management.

## 0) Quick onboarding
If this is your first time in the project, follow this exact order.

### Prerequisites
- Node.js 20+ installed
- npm installed

### Install dependencies
From project root, run:

```bash 
npm install 
```

### Start the API
From project root, run:

```bash
npm start
```

expected output:
```text
Example app listening on port 3000
```

### run:
```bash
npm run test:api
```

expected to run a battery of requests

## 1) Project goal

Build the simplest possible **real-world** authentication + authorization system.
